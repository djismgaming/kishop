# KICart

KICart, a mobile-first web app created to solve the common problem of managing shopping lists and calculating checkout totals with tax in a simple, efficient manner. Keep track of shopping items, quantities, and prices while shopping and calculating tax. Provide a streamlined solution that combines shopping list management with automatic tax calculation in a easy-to-use interface.

## Features

- **Shopping List Management**
  - Add items with quantities and prices
  - Enter prices with standard decimal input
  - Update quantities with a selector with quantities 1 -> 5
  - Edit prices inline
  - Delete items from the list
  - Real-time item count and cost calculation
  - Items without prices are automatically sorted to the top

- **Checkout Calculator**
  - View total item count and subtotal based on actual prices
  - Calculate tax with customizable tax rate
  - Default tax rate value of 11.5%
  - **Persistent tax rate** - your tax rate is saved and remembered
  - See final total with tax included

- **Theme Customization**
  - Light, dark, and system theme options
  - Theme preference is saved and remembered
  - Responsive design that works on mobile devices

- **Data Management**
  - Clear all shopping data with confirmation
  - Persistent data storage
